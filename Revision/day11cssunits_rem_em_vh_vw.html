two categories: absolute units and relative units.

Absolute Units:

Pixels (px):

Pixels are the most commonly used absolute unit in web development. They provide a fixed-size measurement, with one pixel representing one dot on the screen. Pixel units are reliable and consistent across different devices and resolutions, making them a popular choice.

Points (pt):

Points are another absolute unit commonly used in print media but less so in web development. One point is 1/72 of an inch, making it a physical measurement. However, due to the variability of screen sizes and resolutions, points are less predictable than pixels and are used less frequently in web design.

Relative Units:

Percentage (%):

Percentage units are relative to the parent element's size. For example, setting a width of 50% on a container div will make it half the width of its parent element. Percentages are commonly used for responsive design, allowing elements to adapt to different screen sizes and orientations.

Relative Lengths (em, rem):

  • em: One em is equal to the font size of the current element. For example, setting a font size of 2em on a paragraph will make the text twice as large as the font size of its parent element.
  • rem: One rem is equal to the font size of the root element (). Rem units are useful for creating more predictable and consistent layouts, especially in large-scale projects where nesting can become complex.

Viewport Width/Height (vw/vh):

These units are relative to the viewport's width or height. For example, setting a width of 50vw on an element will make it 50% of the viewport's width, regardless of the screen size. Viewport units are commonly used for creating fully responsive designs that adapt well to different screen sizes.

What is it ..?

The full form of VW is viewport width. It works like the percentage unit. Specifying 10vw is equivalent to occupying 10% of entire visible screen width.

The full form of VH is viewport height. It works like the percentage unit as well. Specifying 10vh is equivalent to occupying 10% of entire visible screen height.
